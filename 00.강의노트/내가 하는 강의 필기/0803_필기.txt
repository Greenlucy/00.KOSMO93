< 테이블 용어 설명 > (*는 알고 있어야 함)
테이블 : 관계형 데이터베이스의 기본 데이터 저장 구조로 로우와 컬럼으로 구성되어 있음
	하나의 테이블은 여러개의 로우로 구성되어 있음
*로우 : 테이블을 구성하는 단위. 테이블은 로우 단위로 구성되어 있음. 로우 = 건 수
컬럼 : 하나의 로우를 구성하는 구성요소
프라이머리키 : 테이블에서 각 로우를 유일하게 구분하는 컬럼키 
	NOT NULL (데이터가 꼭 있어야 한다.) + UNIQUE (해당 컬럼에서 값이 유일해야 한다.)
*필드(셀) : 테이블에서 로우와 컬럼이 교차하는 데이터
외래키 : 관계키. 테이블을 연결하는 키 (다른 테이블의 기본키를 참조하는 키)
* 널 : 필드에 값이 없는 것


오라클은 계정 단위로 움직임

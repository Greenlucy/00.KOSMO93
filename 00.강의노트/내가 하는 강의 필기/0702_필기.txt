< 코딩하는 순서 (4가지 규칙)>
1. 클래스의 껍데기는 클래스 키워드(class), 식별자(사용자 정의 클래스 이름), 브레이스 블럭 { } 로 만든다. 
2. 클래스는 변수와 함수로 구성되어 있다.
3. main( ) 함수의 역할 : 콘솔 어플리케이션의 시작점
4. 클래스를 new하면 : new연산자가 하는 일은 생성자를 찾는다.

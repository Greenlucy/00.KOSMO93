키워드(예약어)
	package : 클래스의 묶음 : 자바에서 최고로 큰 단위 
			: *.java 여러개 있음
	class : 클래스를 만드는 키워드 
		일반 클래스 : class 사용자정의 클래스 이름 {}
		인터페이스 클래스 : public inteface 사용자정의 클래스 이름 {}
		추상 클래스 : public abstract class 사용자정의 클래스 이름 {}
		파이날 클래스 : public final class 사용자정의 클래스 이름 {}
	public : 접근제한자 : access modifier
		접급제한자는 class, 변수, 함수 앞에 선언하기 
		class, 변수, 함수 모두 각자에게 권한을 선언해야 한다. 
		public > protected > 아무것도 않쓰기(default) > private 
	static 
		메모리에 올리는 키워드
		class, 변수, 함수 앞에 선언하기 
	void
		리턴 값이 없다 : return 키워드를 함수 블럭에서 사용할 수 없다.
	return
		함수의 결과 값을 호출하는 함수에 전달하기 
	new 
		클래스를 객체로 만드는 연산자 : 클래스를 메모리에 올리는 연산자 

기호
	{} : 블록, scope, 영역
	() : 함수의 매개변수 선언 : 함수를 만들 때 : 데이터 타입 변수명(int i, String s)
	     함수의 아규먼트 선언 : 함수를 사용할 때(호출할 때) : 값 (변수명) : (59, "현병은")
		 함수의 매개변수와 아규먼트는 데이타 타입과, 순서는 같아도, 변수명을 달라도 된다. 
	. : 현재 디렉토리, 링크 연산자(클래스와 함수연결, 함수와 함수 연결)
	; : 문장(statement) 종결

연산자

	[] : 배열 연산자

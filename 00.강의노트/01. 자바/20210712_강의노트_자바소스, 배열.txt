알고리즘 과 자료구조 

알고리즘 
	일반적으로 특정 문제를 해결하기 위한 어떤 동작들의
	단계나 방법의 모임이다.
	
	시간 복잡도 : Time Complexity
		일반적으로 알고리즘에 의해서 수행되는 기본 연산의 횟수를 나타낸다.
		컴퓨터는 1초에 약 1억 번의 기본연산(산술연산, 제어연산)한다고 가정한다.
	공간 복잡도 : Space Complextiy
		메모리의 사용량
		자바 : 힙, 정적 메모리의 합

자바에서 배열 

배열 : 객체 : new : 생성자 : 디폴트값 초기화 
	자료를 순차적(index, 첨자)으로 관리하는 자료구조이다. 
	index, 첨자는 항상 0 부터 시작한다. 
	데이터 타입이 동일 해야 한다. 
		byte[] b = new byte[];
		char[] c = new char[];
		short[] s = new short[];
		int[] i = new int[];
		double[] d = new double[];
		..................
	데이터 타입(자료형)은 
		기초자료형, 참조자료형, 빌트인 클래스, 사용자 정의클래스 ... 
	자바에서 배열은 Object 이다. 
	자바 배열 선언은
		1차원 배열 
			자료형[] 배열 참조변수이름 = new 자료형[개수];
			자료형 배열 참조변수이름[] = new 자료형[개수];
		2차원 배열			
			자료형[][] 배열 참조변수이름 = new 자료형[개수][개수];
			자료형 배열 참조변수이름[][] = new 자료형[개수][개수];
			자료형[] 배열 참조변수이름[] = new 자료형[개수][개수];
	자바 배열 연산자 : [] : 브라켓
	자바 배열 길이를 구하는 필드는 : length
	배열은 요소(element)로 구성되어 있다. 
	배열의 요소 : 자료형[요소갯수] 
	자바 배열의 길이는 : 요소개수 - 1 = length : index 가 0부터 시작 
	배열은 선언과 동시에 자료형의 default value로 초기화 된다.

	배열은 선언과 동시에 default value 로 초기화 되는데
	다른 값으로 초기화 할 수 도 있고
	배열의 요소는 인덱스(첨자)를 이용해서 저장, 조회 할 수 있다.

자바 클래스 
-------------
일반 클래스 : class 클래스이름 {}
인터페이스 클래스 : interface 클래스이름 {}
추상 클래스 : abstract class 클래스이름 {}
파이날 클래스 : final class 클래스이름 {}

변수
-------------
멤버변수(=전역변수) : 변수 선언을 클래스 블럭 안에서, 클래스 안에 있는 함수 블럭 밖에서 
지역변수 : 변수 선언을 함수 안에서 하는 것 
클래스변수 : 멤버변수에 static 키워드를 선언하는 것 


함수 : function : 메소드 : method
-------------

	리턴형 함수이름 ([매개변수가 들어오는 공간]) 

	[접근제한자] [수정자] 리턴형 함수이름 ([매개변수가 들어오는 공간]) {
	
		함수 블럭에 로직 만들기;

		[return];
	}

접근제한자 : Access Modifier : public, protected,아무것도 않쓴 것(default),  private 
	public : 클래스, 변수, 함수 사용 가능 : 패키지를 넘다들면서 사용 가능 
	protected : 다른 패키지에서 상속 관계가 있을 때 사용 <-- 나중에 공부하기 
	default : 클래스, 변수, 함수 사용 가능 : 패키지내 에서만 사용 가능
								       (같은 패키지 내에서 *.java 화일에서는 사용가능) 	
	private : 변수에만 사용하기 : 클래스 내부에서만 사용한다.  

수정자 : static, final
	static : 클래스, 변수, 함수 사용 가능 : 해당 하는 자원을 메모리 올린다.
	final : 클래스, 변수, 함수 사용 가능 : 해당 하는 자원을 변경하지 마세요.

리턴형 : void, 자바 데이터 타입(자료형)
	void : return 키워드를 사용할 수 없다 : 리턴 값이 없다.
	자바 데이터 타입(자료형)
		기초 자료형
			숫자
				정수 : byte, short, int, long : 디폴트 값 : 0
				실수 : float, double : 디폴트 값 : 0.0
			문자
				문자 : char : ' '(빈 문자, 싱글 쿼테이션), '\u0000'
				문자열 : String : 기초 자료형이 아니고 참조 자료형
			논리 : boolean : true, false : 디폴트 값 :  false
		참조 자료형
			Object  : 디폴트 값 : null
			빌트인 클래스 API
				Array, interface ... 
			사용자 정의 클래스 
함수이름 : 식별자 
매개변수 : 매개변수는 있어도 되고, 없어도 된다.
		 데이터 타입 선언 변수명 선언, 데이터 타입 선언, 변수명 선언, ......
		 매개변수는 데이터 타입과, 순서를 지켜야 한다.

		매개변수(parameter, 파라미터) : int a, int b
			public int sum(ina a, int b){
				return a+b;
			}
		인자 : 전달인자(argument, 아규먼트) : 10, 20
			sum(10, 20);

{} : 함수 블럭 : 로직을 만드는 공간
return : 리턴 값이 있는 경우도 있고, 없는 경우도 있다.

클래스 구성원 : 변수, 함수 : 자원 : Resource 
API : 라이브러리 : 클래스 : 빌트인 클래스 
예)	java.lang.String : 빌트인 스트링 클래스 : 스트링 
	java.lang.System : 빌트인 시스템 클래스 : 시스템 

자바 기초에서 공부해야 할 패키지 
java.lang.* 패키지에 있는 클래스 : 숫자, 문자, 문자열 
java.util.* 패키지에 있는 클래스 : 날짜, 데이터 관련
java.io.* 패키지에 있는 클래스 : 파일
java.sql.* 패키지에 있는 클래스 : 데이터 베이스 
java.awt.* 패키지에 있는 클래스 : C/S Swing 인터페이스 프로그램
javax.swing.* 패키지에 있는 클래스 : C/S Swing 인터페이스 프로그램 


API : 클래스 : 
클래스 이름{
	필드
		변수
		public static final byte   MAX_VALUE = 127;
	함수
		생성자 
		메소드(함수)
		public static String toBinaryString(int i) {
			return toUnsignedString0(i, 1);
		}
} 

클래스 구성원 : 변수, 함수 
	 keyword(예약어), 기호, 연산자, 식별자(indentifier), 
	 사용자 정의 클래스, 빌트인 클래스(API)
	
식별자(indentifier) : 개발자가 자바 소스코드를 구성하면서 맘대로 쓸 수 있는 것
	사용자 정의 클래스이름 : 첫번째 글자를 꼭 대문자 쓴다. 
	변수 이름 : 첫번째 글자를 꼭 소문자 쓴다.  
	함수 이름 : 첫번째 글자를 꼭 소문자 쓴다.  
	패키지명 : 전체를 소문자로 쓴다. 
	식별자는 영어로 사용한다. 
	식별자는 숫자로 시작하지 않는다. 

	식별자 규칙은 
		이 클래스가 
		이 변수가
		이 함수가 
		어떻한 역활을 하는지 한번에 알 수 있게 만드게 중요하다.

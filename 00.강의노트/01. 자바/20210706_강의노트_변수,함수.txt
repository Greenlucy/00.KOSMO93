2. 변수

멤버변수 : 클래스 안, 함수 밖에 선언되는 변수
	: 멤버변수는 선언과 동시에 초기화 하지 않는다.
	  생성자가 초기화 시켜준다.

	상수 : public static final 데이터타입 변수명(XXX_XXXX) = 리터럴;
	클래스변수 : static 테이터타입 변수명;
	멤버변수 : 데이터타입 변수명;
	전역변수 : public 데이터타입 변수명;

지역변수 : 함수 블럭에 선언하는 변수 
			함수에서 태어나서 함수에서 돌아가신다.
			지역변수는 함수 밖을 나갈 수가 없다.(함수 밖에서 지역변수를 호출 할 수가 없다.)
			지역변수 값을 함수 밖으로 내보려면 return 키워드를 이용해야 한다. 

			지역변수는 항시 선언과 동시에 초기화 한다. 초기화 할 때 그 데이터 타입의 default value 
	기초자료형 : 정수 : 0 : byte, short, int
	          실수 : 0.0 : float, double
			  문자 : ' ' , '\u0000'
			  논리형 : fasle
	참조자료형 : 문자열 : "", null : String 
			

3. 함수 : 리턴형 함수이름([매개변수])


리턴형이 없는 함수 : void  키워드 사용 : return  키워드가 없는 것 
	static 있는 함수 : 클래스이름.함수이름()
		매개변수가 있는 함수 : 클래스이름.함수이름(테이터타입 변수1, 테이터타입 변수2)
		매개변수가 없는 함수 
	static 없는 함수 : 참조변수 사용하기 : 클래스이름 참조변수 = new 생성자이름(); : 참조변수.함수이름();
		매개변수가 있는 함수 : 참조변수.함수이름(테이터타입 변수1, 테이터타입 변수2) 
		매개변수가 없는 함수 

리턴형이 있는 함수 : void 키워드가 없고, 데이터 타입 선언 된 것 : return 키워드가 있는 것 
	static 있는 함수  : 클래스이름.함수이름()
		매개변수가 있는 함수  : 클래스이름.함수이름(테이터타입 변수1, 테이터타입 변수2)
		매개변수가 없는 함수 
	static 없는 함수 : 참조변수 사용하기 : 클래스이름 참조변수 = new 생성자이름(); : 참조변수.함수이름();
		매개변수가 있는 함수  : 참조변수.함수이름(테이터타입 변수1, 테이터타입 변수2)
		매개변수가 없는 함수

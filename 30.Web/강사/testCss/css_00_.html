CSS : Cascading Style Sheets
HTML와 같이 CSS는 실제로 프로그래밍 언어 도 마크업(markup) 언어도 아닌
Style sheet 언어이다.
HTML 문서에 있는 요소들에 선택적으로 스타일을 적용할 수 있다.
HTML 페이지에서 모든 문단 요소들을 선택하고 그 문단 요소들 안에 있는
텍스트를 빨갛게 바꾸려면 다음과 같이 CSS를 작성하면 된다.
p{ color: red; }

인라인 스타일(Inline Style) :

한 줄짜리 짤막한 스타일 태그 안에 직접 지정하여 사용. HTML과 섞어서 사용
<p style='background-color:#00f; color:ff0'>즐거운 하루!


내부 스타일 시트(Internal Style Sheet) :

HTML 파일 안에 별도 영역으로 스타일 정의
일반적으로 <head>...</head> 안에 style을 사용
HTML 문서 하나가 고유한 CSS 내용을 가진다고 생각될 때 사용
<style> 태그를 사용하여 지정
HTML 페이지 안에 위치하므로 영향을 주는 범위는 페이지 안의 전체 태그
다른 HTML 페이지에서 똑같은 내용으로 사용하고 싶다면 복사해서 사용
<head>
<style type='text/css'>
h1{font-family:'굴림체'
font-size:30px; }
</style>
</head>

외부 스타일 시트(External Style Sheet) :

스타일 내용을 별도의 페이지로 만들어서 사용
여러번 복사해야 하는 내부 스타일 시트의 단점 보완
확장자는 .css
하나의 스타일로 여러 개의 HTML 페이지에 적용하여 사용하고자 할 때 편리
하나의 스타일 페이지만 바꾸면 연결된 전체 HTML 페이지가 바뀌므로 유지, 보수에 편리
각각의 HTML 페이지 문서는 <head>...</head> 안에 <link> 태그를 사용하여 스타일 페이지와 연결
<head>
<link rel='stylesheet' type='text/css' href='외부스타일시트명.css'>
</head>

Selector(선택자)

id attribute 선택자 : 엘리먼트 하나에만 적용
class attribute 선택자 : 엘리먼트 여러개 적용

CSS Attributes

style attribute : 엘리먼트에서 인라인 스타일 적용
<style> : internal CSS 적용
<link> : external CSS file 불러오기
<head> : <style> , <link> elements 사용
color property : 텍스트 컬러 적용
font-family property : text fonts 적용
font-size property : text sizes 적용
border property : borders에 적용
padding property : 보더와 텍스트 사이 padding(스페이스) 적용
margin property :보더와 밖같쪽 margin(스페이스) 적용

HTML 문자 엔티티

< : 오른쪽이 열린 꺽쇠 : 태그를 여는 꺽쇠
> : 왼쪽이 열린 꺽쇠 : 태그를 닫는 꺽쇠
<head><title>타이틀 </title></head>
  : 공백문자 한칸 의미, Space 한칸
& : 앰퍼샌드(Ampersand) 문자
" : 쌍따옴표 하나

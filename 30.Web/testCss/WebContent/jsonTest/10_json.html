전역변수(Global Variable) vs 지역변수(Local Variable)
전역변수 : 함수 외부에서 선언된 변수, 프로그램 전체에서 접근할 수 있는 변수
지역변수 : 함수 내부에서 선언된 변수, 함수가 실행되면 만들어지고 함수가 종료되면 소멸되는 변수, 함수 외부에서 접근할 수 없음

변수 abc0의 값을 함수 외부에서는 abc0, 함수 내부에서는 def0으로 초기화
함수 외부에서 abc0에서 호출했으므로 전역변수 값이 호출된다.
abc0

함수 내부에서 abc1의 값을 호출했으므로 지역변수 값이 출력된다.
def1

함수 내부에서 var를 사용하지 않고 변수의 값을 정하면 전역변수의 값을 변경한다.
58라인에 선언한 abc2 = "def2" 변수가 전역변수 역할을 함., var를 선언하지 않고 사용하는 변수
def2

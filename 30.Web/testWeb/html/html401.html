이 문서는 HTML4.01 HTML 문서이다.
HTML5.0과 HTML4.01 문서는 DOCTYPE으로 구분한다.


HTML5.0 독타입 DOCTYPE 은
<!DOCTYPE html>

HTML4.01 독타입 DOCTYPE 은
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtdquot">
우리는 HTML5.0 DOCTYPE을 사용합니다.
strict.dtd : 태그 문법이 엄격하다.
loose.dtd : 태그 문법이 느슨하다.
frameset.dtd : 프레임을 나누어서 만들어준다.
쿼그모드 (Quirks mode) 웹 페이지의 하위 호환성 유지를 시킨다.
예) IE 11에 맞춰 개발했는데, IE6에 알아서 호환된다.
